# 09. Логики — реестр переиспользуемых бизнес-логик

> Раздел содержит переиспользуемые бизнес-логики (ТЗ) клиентского web-приложения кулинарной
> студии «Шеф-стол» (роль **Клиент**). Каждая логика описывает поведение, применимое на нескольких
> экранах: флоу (mermaid), пошаговое описание, таблицы API-запросов, связанные требования и
> критерии приёмки.
>
> **Источники контракта:** API — многофайловый OpenAPI в [`../../api`](../../api) (домены `auth`,
> `bookings`, `slots`, `catalog`, `profile`); требования — [`../../2-requirements`](../../2-requirements);
> экраны — соседние файлы `../SCR-XX_*.md`.
>
> Ссылки на операции API даются как `[../../api/{домен}/api.yaml]` → `operationId`.
